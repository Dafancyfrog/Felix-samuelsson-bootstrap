The foundations of HTML and CSS


Css och HTML grunder med ett anslag av viktiga komponenter som behövs för att jobba med språken



Element

Ett element använder man för att skriva sin kod i html och css. Det defineras med symbolerna "<"och ">"och en kallas en tag medans ett element är allting mitt imellan start tag och slut tag. Dessa används i för allt till exempel "body" elementet gör upp hela kroppen för html documentet. Element bygger upp allting man gör med html. Som exempel ett p element används för att skriva detta.

HTML

Hypertext Markup Language(HTML) är ett programeringspråk för document som ska visas upp i form av webbrowsers. HTML är specifikt gjort för det och det finns ett antal av varianter av språket men vanligtvis är språket använt i samarbete med Cascading Style Sheet(CSS) och JavaScript.

ID

ID är en del av HTML och används för att ge ett element ett unikt id. Id måster vara unikt innom HTML documentet och används på många sätt när man vill välja ett specifikt element. Id kan också användas av JavaScript .

Class

Class elementet liknar id elementet men till skillnad från id elementet så behöver class elementet inte vara unikt utan kan bli delat av oändligt många element. Detta kan användas på flera sätt till exempel om man vill ha containers som jag har på denna sidan så använder man sig av Class "container" i bootstrap för att alla ska bli lådor.

Meta

Meta elementet är en förkortning för metadata och används i htmml document. Metadata är data(information) av data. En meta tag används i head elementet och kan användas för att ändra layouten och storleken av sidan så man kan använda den på telefånen eller en tablet utan att den ser konstig ut. Metadatan kommer inte ses på sidan utan används av machinerna bakom allt.

Responsive

Att en hemsida är responsive betyder att hemsidan kommer automatiskt att justera sig beroende på vad man använder varesig tablet, desktop eller telefån så kommer blider och text ändra storlek så det passar med vad du använder. Detta är gjort med hjälp med meta elementet som justerar storleken på allt.

http

http ller Hypertext Transfer Protocol är ett kommunikations protokoll som grundades för att kunna leverera HTML sidor från webservrar till web klienter. I dagens samhälle används det för att överföra webbsidor till The World Wide Web (WWW)

https

Hypertext Transfer Protocol Secure eller https är liknande med http med att det är ett sätt att skicka html sidor. Men det tillagda S i slutet av orddet betyder Secure och det krypterar medelandet vi vill skicka så att det är svårare att använda om man får tag på det och inte kan kryptera det tillbaka. Detta hjälper med säkerheten på webben och är väldigt bra att ha.

Compressed data

Compressing data är när man har en bit kod men skriver om den så den ska ta mindre plats och vara mera effektiv. alltså man förkortar den så mycket som möjligt så datorerna kan ladda den snabbare och utan problem

CDN

CDN eller Content Delivery Network är en stor grupp sv utspridda servrar över flera platser på jorden. Dessa servrar sparar kopior av data så att man kan enklare och snabbare få data oberoende vart man bor eftersom servrarna finns på flera olika platser så skickas man datan av den som är närmast och detta hjälper med att minska laddningstiden och gör data enklare och snabbare att få tag på. .

Local Bootstrap

Local bootstrap kan användas av CDN och är bara vanliga bootstrap som är närmare kan göra sidorna använder bootstrap att ladda snabbare och kan till och med minska bandwidth användningen av sida på grund av sättet CDN skickar data. Detta är viktigt för att hosta en websida kostar pengar och vissa hosting företag tar betalt på hur mycket bandwidth du använder så det kan minska priset på hela sidan.

JQuery

JQuery är ett JavaScript biblotek av kod som är tänkt att förenkkla HTML och css användning. JQuery liknar bootstrap på många sätt med att det hjälper att dra ner tiden att göra en hemsida med efetrsom mycket av det man använder är redan gjort och man behöver bara ändra lite istället för att göra zin sida från grunden.

Flexbox

Flexible box moduel eler flexbox kan man gissa vad det är med tanke på att box är i namnet. Men som du kan ha gissat så hjälper flexbox att göra boxes i html och css vilket används mycekt i webdesign och förenklar skapandet och användnignen av dessa boxes.

Grid

Grid systemet är en avd de kraftfullaste layout systemet i css eftersom till skillnad från flexbox som bara fungerar på 1 diension så fungerar gridsystemet på 2 dimensioner. Detta används mycket eftersom det oftast hjälper med att designa websidor utan att behöva används saker som positioning och floats som sparat mycket tid och plats som koden tar upp.

Codesnippets

Code snippets är korta bitar av kopierbar kod som kan återanvändas på många sätt. Man brukar använda code snippets genom att man kopierar in dessa bitar av kod i större program för att göra programeringen enklare.
